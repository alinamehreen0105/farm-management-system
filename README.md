# Farm Management System

This project is a **Farm Management System developed using Salesforce Developer Org**.

## Overview

The system helps manage farm-related information such as farmers, farms, crops, and buyers in a centralized platform.

## Objects Used

* Farmer
* Farm
* Crop
* Buyer

## Features

* Manage farmer details
* Track farm and crop information
* Manage buyer data
* Generate reports and dashboards
